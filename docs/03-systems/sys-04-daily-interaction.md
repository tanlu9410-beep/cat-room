# sys-04 日常互动系统

## 这份文档是干什么的

记录玩家与场景内实体的日常互动规则。  
当前已实现的互动主要是鼠标操作。

## 当前已实现的互动

### 鼠标操作层级

点击时按以下优先级检测命中对象：

1. **机器人**（最高优先）
2. **猫**
3. **纸团**
4. **家具**（最低优先）

### 机器人互动

| 机器人状态 | 点击效果 |
|-----------|---------|
| stuck / frenzy | 修复：恢复 idle，吐出金色纸团 |
| 其他正常状态 | 85% 概率抓起拖拽；15% 概率弄翻（进入 stuck） |
| 抓起后放下 | 恢复 idle |

### 猫互动

- 点击抓起，拖拽移动
- 大体型猫（黑/灰）命中范围更大（60px vs 40px）
- 抓起时猫四肢乱蹬动画
- 放下时：
  - 如果附近有空闲的猫（<60px），触发嗅闻 → 50% 理毛 / 50% 追逐
  - 否则恢复漫步

### 纸团互动

- 点击普通纸团：弹出冷知识卡片（5 秒后自动消失）
- 点击金色纸团：弹出机器人日记卡片
- 卡片位置自适应屏幕边界

### 家具互动

- 点击抓起，拖拽移动
- 拖拽使用偏移量（ox/oy）保持抓取手感
