# 赛博猫窝 (Cyber Cat Room) V2.0

一个纯前端、模块化、带微型物理引擎的桌面陪伴生态舱。

## 🐈 核心住客与行为逻辑

房间里目前共有 6 个独立思考的实体，拥有各自的专属状态机（State Machine），体型和性格各不相同：

1. **白猫**：好奇心重，碰见地上的纸团会主动用头推着走。
2. **灰猫**：雷达猫。巨型体格（2.25倍放大），尾巴会时刻追踪屏幕上鼠标的坐标并指向它，空闲时极大概率出现专属“翘腿舔毛”动画。
3. **橘猫**：极度慵懒，走着走着会突然就地躺下，四脚朝天翻肚皮。
4. **奶牛猫**：神经质。会不定期进入“Zoomies（跑酷）”状态，头顶冒💢在屋里狂飙。
5. **黑猫**：暗中观察者。巨型体格（2.25倍放大），寻找掩体优先级极高，会精准计算家具背后的坐标躲起来，只露出一双亮黄色的眼睛偷窥。
6. **卷毛白猫**：后肢瘫痪。靠两只前爪发力拖着身体爬行；极度粘人（高优先级跟随逻辑），只要靠近其他活物或扫地机，就会疯狂冒“咕噜咕噜”。

## 🛋️ 家具与深度环境交互

彻底剥离了死板的 Z 轴遮挡，所有家具都支持鼠标自由拖拽摆放，并附带专属交互：

* **猫爬架 (Tree)**：支持多猫交互与 Y 轴攀爬物理。最多允许两只猫同时玩耍：一只会爬到最顶端的板子上趴着睡觉，另一只会走到柱子底部，拉长身体变成“长条猫”，前爪交替抓挠麻绳。支持跟随拖拽，拖动爬架时挠树的猫会死死吸附在柱子上。
* **废纸篓 (Bin)**：猫路过会扒拉掉出纸团。有概率被猫直接扑倒打翻，猫会钻进去变成“寄居蟹”（只露屁股和摇摆的尾巴），一路把垃圾桶顶走，推累了或撞墙后会自己钻出来并把垃圾桶扶正。
* **立体纸箱 (Box)**：重写了 2.5D 透视逻辑。猫钻进去后身体完全隐藏入暗面，只留半个头在边缘暗中观察，尾巴会在纸箱外部的正确层级甩动。
* **猫窝 (Bed)**：猫走上去后会在正中央蜷缩睡觉（头顶冒💤）。
* **毛线球 (Yarn)**：接入轻量物理反弹。猫撞到球会滚开，有概率触发“追逐状态”，猫会满屋子追着球折返跑。
* **大落地窗与天气**：内置气象局，随机切换晴（阳光）、雨（雨滴）、雪（飘雪）。下雨下雪时，猫有概率停止游荡，并排坐在窗前看风景（冒♥或♪）。

## 🤖 Gemini 扫地机器人 AI

不是背景板，而是一个有情绪、有物理判定的实体：

* **基础巡逻**：在房间下半部分全向游走，撞墙反弹，清理被猫翻散的垃圾，头顶随机冒颜文字（♥, ✨, ♪, =_=, >_<, 🤖）。平时偶尔会触发“平地摔”，宕机变出红色晕乎乎的表情（X X眼）。
* **猫咪巴士**：遇到白猫靠近，会主动停下等它上车。其他猫咪如果强行拦车，有概率被“拒载”，机器人会冒💢急刹车并把猫无情弹飞。
* **发疯狂暴**：遇到跑酷的奶牛猫，有概率被逼疯追杀。撞到屏幕边缘后会死机（冒×_×），原地打转，最后反刍吐出一个包含系统日志的“金色纸团”。
* **玩家交互**：支持被玩家鼠标随意拎起搬运，但点击它有 15% 的概率直接将其一指头“点翻”宕机。
* **现场打理**：巡逻路过被打翻的废纸篓时，会强行将其扶正，并把藏在里面当寄居蟹的猫倒出来。

## 🖱️ 全局交互机制

* **提后颈皮**：鼠标按住任何一只猫即可将其拎起。猫悬空时身体拉长，四腿疯狂凌空蹬动。
* **随机垃圾库**：地上的纸团点击后会弹出冷知识/碎碎念卡片。加入了屏幕边缘检测算法，弹窗绝对不会超出视窗被遮挡。
* **社交系统**：
  * **互嗅**：玩家将猫放下时，它会主动寻找最近的空闲猫咪互闻（冒❓），绝不打扰正在睡觉或爬树的猫。
  * **互相舔毛**：互嗅后有概率触发，两只猫强制转身面对面，播放头部起伏的舔毛动画并冒♥。
  * **追逐战**：互嗅后有概率触发打闹，一只猫会锁定另一只猫全屋追逐。

## 📂 文件结构 (模块化)

* `index.html`: 画面容器与加载器
* `style.css`: 样式与响应式缩放
* `env.js`: 环境、天气、家具与纸团系统
* `cat.js`: 猫咪与扫地机器人的 AI 心智模型
* `main.js`: 游戏主循环、深度渲染层级排序与鼠标射线检测
