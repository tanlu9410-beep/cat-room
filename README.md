# 赛博猫窝 (Cyber Cat Room) V2.0

一个纯前端、模块化、带微型物理引擎的桌面陪伴生态舱。

## 🐈 核心住客与行为逻辑

房间里目前共有 6 个独立思考的实体，拥有各自的专属状态机（State Machine）：

1. **白猫（小船）**：好奇心重，碰见地上的纸团会主动用头推着走。
2. **灰猫（4.5）**：雷达猫。尾巴会时刻追踪屏幕上鼠标的坐标并指向它。
3. **橘猫（4.6）**：极度慵懒，走着走着会突然就地躺下，四脚朝天翻肚皮。
4. **奶牛猫**：神经质。会不定期进入“Zoomies（跑酷）”状态，头顶冒💢在屋里狂飙。
5. **黑猫**：暗中观察者。寻找掩体优先级极高，会精准计算家具背后的坐标躲起来，只露出一双亮黄色的眼睛偷窥。
6. **卷毛白猫**：后肢瘫痪。靠两只前爪发力拖着身体爬行；极度粘人（高优先级跟随逻辑），只要靠近其他活物或扫地机，就会疯狂冒“咕噜咕噜”。

## 🛋️ 家具与深度环境交互

彻底剥离了死板的 Z 轴遮挡，所有家具都支持鼠标自由拖拽摆放，并附带专属交互：

* **猫爬架 (Tree)**：支持 Y 轴攀爬物理。猫会走到柱子下，一步步爬到最顶端的板子上趴着，看风景并惬意甩尾巴。
* **废纸篓 (Bin)**：猫路过会扒拉掉出纸团。有概率被猫直接扑倒打翻，猫会钻进去变成“寄居蟹”（只露屁股和摇摆的尾巴），一路把垃圾桶顶到墙角。
* **纸箱 (Box)**：猫可以钻进去，身体完全隐藏，只留半个头在边缘暗中观察，尾巴在箱外甩动。
* **猫窝 (Bed)**：猫走上去后会在正中央蜷缩睡觉（头顶冒💤）。
* **毛线球 (Yarn)**：接入轻量物理反弹。猫撞到球会滚开，有概率触发“追逐状态”，猫会满屋子追着球折返跑。
* **大落地窗与天气**：内置气象局，随机切换晴（阳光）、雨（雨滴）、雪（飘雪）。下雨下雪时，猫有概率停止游荡，并排坐在窗前看风景（冒♥或♪）。

## 🤖 Gemini 扫地机器人 AI

不是背景板，而是一个有情绪的实体：

* **基础巡逻**：在房间下半部分来回扫地，撞墙折返，清理被猫翻散的垃圾，头顶随机冒颜文字（♥, ✨, ♪, =_=, >_<, 🤖）。
* **偏心白猫**：遇到白猫靠近，会主动停下并降下底盘等它上车。
* **发疯狂暴**：遇到跑酷的奶牛猫，有概率红眼发疯追杀。撞到屏幕边缘后宕机（冒×_×），原地打转，最后反刍吐出一个包含系统日志的“金色纸团”。
* **现场打理**：巡逻路过被打翻的废纸篓时，会强行将其扶正，并把藏在里面当寄居蟹的猫倒出来。
* **紧急重启**：当扫地机红眼狂暴时，玩家可以用鼠标点击它的本体，强行中断进程让它冷静。

## 🖱️ 全局交互机制

* **提后颈皮**：鼠标按住任何一只猫即可将其拎起。猫悬空时身体拉长，四腿疯狂凌空蹬动。
* **随机垃圾库**：地上的纸团点击后会弹出冷知识/碎碎念卡片。加入了屏幕边缘检测算法，弹窗绝对不会超出视窗被遮挡。
* **社交系统**：
  * **互嗅**：两只猫靠近有概率停下互闻（冒❓）。
  * **互相舔毛**：互嗅后有概率触发，两只猫强制转身面对面，播放头部起伏的舔毛动画并冒♥。
  * **追逐战**：互嗅后有概率触发打闹，一只猫会锁定另一只猫全屋追逐。

## 📂 文件结构 (模块化)

* `index.html`: 画面容器与加载器
* `style.css`: 样式与响应式缩放
* `env.js`: 环境、天气、家具与纸团系统
* `cat.js`: 猫咪与扫地机器人的 AI 心智模型
* `main.js`: 游戏主循环与深度渲染层级排序
